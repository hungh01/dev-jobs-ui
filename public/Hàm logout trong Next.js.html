Dưới đây là cách bạn có thể viết hàm logout ở phía UI (client-side) trong một ứng dụng Next.js sử dụng app directory và kết hợp với API route hoặc Server Action như bạn đã cho ở trên.

✅ Cách 1: Gọi trực tiếp server action từ UI (nếu dùng server action)

app/logout-button.tsx

tsx
"use client";

import { logout } from "./actions/logout"; // đường dẫn đến file chứa server action
import { useTransition } from "react";

export default function LogoutButton() {
    const [isPending, startTransition] = useTransition();

    return (
        <button
            onClick={() => startTransition(() => logout())}
            disabled={isPending}
            className="px-4 py-2 bg-red-500 text-white rounded"
        >
            {isPending ? "Đang đăng xuất..." : "Đăng xuất"}
        </button>
    );
}

✅ Cách 2: Gọi API /api/logout từ UI

Nếu bạn không muốn dùng server actions, bạn có thể tạo một API Route để xoá cookie và redirect.

pages/api/logout.ts

ts
import { NextApiRequest, NextApiResponse } from "next";

export default function handler(req: NextApiRequest, res: NextApiResponse) {
    res.setHeader("Set-Cookie", "Authentication=; Max-Age=0; Path=/; HttpOnly");
    res.status(200).json({ message: "Logged out" });
}

UI client component:

tsx
"use client";

import { useRouter } from "next/navigation";

export default function LogoutButton() {
    const router = useRouter();

    const handleLogout = async () => {
        await fetch("/api/logout", { method: "POST" });
        router.push("/auth/login");
    };

    return (
        <button onClick={handleLogout} className="px-4 py-2 bg-red-500 text-white rounded">
            Đăng xuất
        </button>
    );
}

📁 Cấu trúc

bash
app/
├─ actions/
│  └─ logout.ts       ← Server action để logout

📌 1. app/actions/logout.ts (Server Action)

ts
"use server";

import { cookies } from "next/headers";
import { redirect } from "next/navigation";

export async function logout() {
    const cookieStore = cookies();
    cookieStore.delete("Authentication"); // Xóa cookie auth

    // Chuyển hướng về trang login
    redirect("/auth/login");
}

cookies() trong Server Action không cần await.

📌 2. app/components/LogoutButton.tsx (Client Component)

tsx

    const handleLogout = () => {
        startTransition(() => {

            disabled={isPending}

✅ locator() là gì?

locator() là hàm được dùng trong Playwright (không phải Puppeteer gốc), để tìm phần tử DOM trên trang.

ts
await page.locator('.devsite-result-item-link').click();

Trong dòng đó:

  • .devsite-result-item-link là selector — cụ thể là: 👉 CSS selector, chọn theo class (dấu . đứng trước).

🧠 Vậy locator('...') có thể dùng gì?

locator('selector') trong Playwright hỗ trợ nhiều kiểu selector:

Loại SelectorVí dụGhi chú
CSS.class, #id, div > spanCực kỳ phổ biến
Texttext="Đăng nhập"Tìm theo text trên trang
XPathxpath=//div[@class="title"]Dùng khi CSS không đủ linh hoạt
RolegetByRole('button', { name: 'Submit' })Hữu ích cho test accessibility
Customdata-testid="my-button"Khi dùng attr riêng cho test

🧪 Ví dụ thêm

ts
// Theo class
await page.locator('.btn-primary').click();

// Theo text
await page.locator('text="Đăng xuất"').click();

// Theo attribute
await page.locator('[data-testid="logout-button"]').click();

🔁 So sánh với Puppeteer

Trong Puppeteer, bạn dùng:

ts
await page.click('.devsite-result-item-link');

hoặc

ts
const el = await page.$('.devsite-result-item-link');
await el?.click();

const puppeteer = require('puppeteer');

(async () => {
  const browser = await puppeteer.launch({
    headless: true, // đổi false để xem trình duyệt
    defaultViewport: null,
  });

  const page = await browser.newPage();
  await page.goto('https://www.topcv.vn/tim-viec-lam-fullstack-developer', {
    waitUntil: 'networkidle2',
  });

  await page.waitForSelector('.job-item-search-result');

  const jobs = await page.$$eval('.job-item-search-result', (jobCards) =>
    jobCards.map((card) => {
      const title = card.querySelector('h3.title a span')?.innerText.trim();
      const company = card.querySelector('.company-name')?.innerText.trim();
      const salary = card.querySelector('.title-salary')?.innerText.trim();
      const location = card.querySelector('.label-content .address .city-text')?.innerText.trim();
      const experience = card.querySelector('.label-content .exp span')?.innerText.trim();
      const tags = Array.from(card.querySelectorAll('.tag a')).map((el) => el.innerText.trim());
      const link = card.querySelector('h3.title a')?.href;
      const image = card.querySelector('.avatar img')?.src;

      return { title, company, salary, location, experience, tags, link, image };
    })
  );

  console.log(jobs);

  await browser.close();
})();
